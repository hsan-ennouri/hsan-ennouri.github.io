IoT-Enabled Biometric Smart Vault
Fall 2024-2025 (NUS - CS3237)
Objective
Co-developed a secure, internet-connected physical vault. Responsible for the complete mechanical design, hardware integration, and the local embedded firmware.
Key Contributions
  • Mechanical Design (CAD): Designed the vault enclosure and a custom sliding-bolt locking mechanism using CATIA. Modeled internal components to securely house the electronics and servo motor.
  • Embedded Firmware: Developed C++ control logic handling the 4x4 Keypad input, LCD status display, and Servo actuation. Implemented state machines for system initialization and sleep modes.
  • Hardware Integration: Integrated the ESP32 microcontroller with peripherals (servo, buttons, LCD), ensuring reliable physical actuation based on sensor inputs.
